# ichiyasaGitSample
『いちばんやさしいGit&GitHubの教本』のサンプルプロジェクト

## 改善メモ

### コンテンツ面

1. **スピーカー情報が未定** - `1人目: 未定` / `2人目のプロフィール` などプレースホルダーのまま。「TBA（後日公開）」など丁寧な表現に変更すべき
2. **セッション内容が未定** - タイムテーブルの「セッション1: 未定」「セッション2: 未定」も同様
3. **参加申し込みリンクがない** - connpassやDoorkeeper等へのCTAボタンが必要
4. **年が不明** - 「3月23日」のみで年が記載されていない
5. **グループ説明が薄い** - コミュニティ説明が3行のみで活動の魅力や対象者が伝わらない
6. **コピーライト表記が未更新** - `© Untitled. All rights reserved.` はテンプレートのまま
7. **ナビのラベルの不一致** - ナビには「Japan Git User Group」、セクションには「Ichiyasa Git User Group」と名称がずれている

### SEO・アクセシビリティ面

8. **`<meta description>` がない** - 検索エンジン向けの説明文が未設定
9. **画像の `alt` 属性が空** - `avatar.png`、`speaker1.png`、`speaker2.png` すべて `alt=""` でスクリーンリーダー非対応
10. **OGPタグがない** - SNSシェア時にタイトル・画像・説明が表示されない

### 画像ファイル

11. **`speaker1.png` / `speaker2.png` が存在しない** - HTMLで参照されているが `images/` に存在せず、壊れた画像アイコンが表示される
12. **`banner.png` が793KBと巨大** - WebP変換＋圧縮で100KB以下を目標にすべき

### CSS

13. **Google Fontsを `@import` で読み込んでいる** - レンダリングブロッキングになるため `<link>` タグに変更すべき
14. **未使用のグリッドクラスが大量にある** - 3094行のうち大半が未使用。PurgeCSSなどで削除すると大幅に軽量化できる
15. **`font-awesome.min.css` を読み込んでいるがアイコンを使っていない** - 不要なら削除すべき

### JavaScript

16. **`skel.min.js` が古い** - メンテされていない旧フレームワーク。モダンなCSSのメディアクエリで代替可能
17. **jQuery依存が重い（94KB）** - スムーススクロールとナビのアクティブ強調のためだけに使用。バニラJSで代替できる
18. **スクリプトに `defer` / `async` がない** - `</body>` 直前でも `defer` を明示するのがモダンな書き方

### その他

19. **`.gitignore` が最低限すぎる** - `node_modules/`、`*.log`、`.env` などの一般的な除外が未設定
20. **README の説明が1行のみ** - セットアップ手順やスクリーンショットがあるとより親切
